# Prompts

Store reusable task-context templates here. Keep prompts provider-neutral and
keep execution behavior in `roles/` strategy profiles.

- `executor-soul.md`: constrained task-specific behavior proposal; authority
  remains in the generated executor's `meta.json`.
- `native-subagent-brief.md`: compatibility template for a bounded native
  subagent brief; prefer `executor-soul.md` for executor work.
